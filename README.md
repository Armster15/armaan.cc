# armaan.cc

My personal website, which includes a blog, my projects, and information on who I am.

The first commit in this repository was back in November 2023, which marks when I moved my personal website to Astro. I then redesigned it in April 2025.

My general philosophy for this codebase is to keep it as maintainable as possible for the forseeable future. Therefore, I aim to minimize the amount of lines of code and third-party dependencies as much as I can. While LOC is a silly metric, it works as a decent measure for this case. I'm particularly proud of the fact that the most used programming language in this codebase is MDX, which is just markup of the blog posts. And while in theory the best way to ensure long-term compatibility is to use plain HTML/CSS/JS instead of a JavaScript framework, I don't want maintaining this to be annoying: good devex is important!

On the contrary, I also do enjoy playing around with new tools and features, and so I'll still use this codebase as a living playground to try them out.

Nearly all of the code here is hand-written. However, I'd like to strongly emphasize that this codebase doesn't have a pro/anti-AI stance: rather, it stems from the fact that I _do_ enjoy writing code and find it relaxing. Maintaining this website is my way of having fun and giving myself a break from other more serious things in my life. Therefore, while most of the code is written manually, for more tedious or annoying things I'll probably use AI.

Blog posts will always be written by me and never by AI. As writing _is_ the output, it must be composed by a human.
